# 沐星人 MuStar｜外星電影品牌・商業版（GitHub Pages）

## ✅ 這包包含
- 商品展示（可篩選）
- 前端購物車（LocalStorage）
- 一鍵用 Line 帶單（自動組合訂單訊息）
- 手機優化、速度優化（星空是輕量版）

## 🚀 上線方式
1. 解壓縮
2. 把檔案上傳到 GitHub repo 根目錄（index.html 要在最外層）
3. GitHub → Settings → Pages → Branch 選 main / root → Save
4. 網址：https://<你的帳號>.github.io/<repo>/

## ✏️ 你最常改的地方
- Line ID：`js/main.js` 裡的 `LINE_ID`
- 商品內容/價格：`js/main.js` 的 `products`
- 免運/運費：`FREE_SHIPPING_AT` / `SHIPPING_FEE`
