# 沐星人 MuStar｜外星電影品牌（商業版）

這是一個可直接上 GitHub Pages 的靜態官網：
- 電影級外星視覺（但手機優化、不卡）
- 內建購物車（localStorage）
- 一鍵複製「下單訊息」到剪貼簿 → 貼到 Line 下單

## 上線方式（GitHub Pages）
1. 解壓縮 ZIP
2. 把檔案（index.html、css、js、assets）上傳到 repo 根目錄
3. GitHub → Settings → Pages → Branch 選 main → Save
4. 網址： https://<你的帳號>.github.io/<repo>/

## 修改 Line ID
`js/main.js` 內的 `LINE_ID` 改成你的 Line。
